## 1) نظرة عامة على مركز المساعدة
- مركز مساعدة احترافي يخدم جميع مستخدمي منصة إدارة المدارس متعددة المدارس.
- الهدف: تقديم أدلة شاملة، بحث سريع، تصدير PDF، وفيديوهات تعليمية، مع سهولة تحديث المحتوى.

## 2) الأقسام المطلوبة
- دليل مدير المنصة (Platform Admin)
- دليل مدير المدرسة (School Admin)
- دليل المعلم
- دليل ولي الأمر
- دليل الموظفين (المالية – شؤون الطلاب – شؤون الموظفين – الموارد البشرية)
- الدليل المالي
- دليل الحضور والغياب
- دليل الجداول
- دليل الرسائل والإشعارات
- التقارير والتحليلات
- الأسئلة الشائعة FAQ
- كيفية البدء Get Started
- فيديوهات تعليمية (Video Tutorials)

## 3) المتطلبات الوظيفية
- عرض ملفات Markdown لكل قسم داخل `/help-center`.
- بحث داخلي فوري عبر عناوين الأقسام والمحتوى.
- تصدير أي دليل إلى PDF بجودة عالية ودعم RTL.
- دعم الفيديوهات عبر تضمين الروابط أو قوائم تشغيل.
- Tooltips تفاعلية ضمن صفحات رئيسية (طلاب/معلمين/الحضور…).
- تبديل اللغة (عربية/إنجليزية) من شريط اللغة.

## 4) المتطلبات غير الوظيفية
- قابلية التوسّع: إضافة/تحديث المحتوى دون تغييرات في الكود.
- سهولة الصيانة: بنية ملفات واضحة وموحّدة.
- الأداء: فهرس بحث محلي فعال.
- الأمان: احترام RBAC في عرض الأدلة المتقدمة.

## 5) قصص المستخدم
- كمدير منصة: أرغب في أدلة إعداد وتكوين النظام، وتصديرها PDF.
- كمدير مدرسة: أرغب في خطوات إدارة الطلاب والمعلمين والمالية.
- كمعلم: أرغب في فهم الجداول والحضور والرسائل.
- كولي أمر: أرغب في معرفة الوصول للبوابة ومتابعة الطالب.
- كموظف مالي/شؤون/HR: أرغب في إجراءات التشغيل ونماذج العمل.

## 6) التدفقات الأساسية
- الدخول إلى `/help-center` → اختيار القسم → قراءة المحتوى → بحث/تصدير.
- ضغط زر “تصدير PDF” → طلب خلفي → تنزيل الملف.
- مشاهدة فيديو تعليمي → فتح رابط خارجي أو تضمين داخل الصفحة.

## 7) إدارة المحتوى
- المجلدات:
  - `frontend/docs/` للأدلة الظاهرة للمستخدمين.
  - `backend/docs/` لمحتوى تقني وإداري.
- إضافة ملف جديد: `*.md` باسم واضح، وتحديث فهرس الأقسام.
- بنية موحّدة للعناوين: مقدمة، خطوات عملية، أسئلة شائعة، روابط مفيدة.

## 8) البحث والتصنيف
- يقرأ العميل قائمة الملفات ويُبنى فهرس مبسّط (العناوين/الملخص).
- البحث يدعم العربية بدون تشكيل ويعرض النتائج بترتيب الصلة.

## 9) PDF والتوافق مع العربية
- تحويل Markdown → HTML → PDF مع هوامش وترويسات.
- يدعم RTL، خطوط عربية مناسبة.

## 10) Tooltips التفاعلية
- نقاط مساعدة داخل الواجهة على عناصر رئيسية.
- قابلية التفعيل/التعطيل لكل دور.

## 11) قياس الأداء والاستخدام
- عدادات زيارة الأدلة، الزمن داخل الصفحة، أكثر الأقسام استخدامًا.
- يساعد على تحسين المحتوى واستهداف مناطق الفجوة.

## 12) خارطة طريق قصيرة
- الإصدار الأول: الأدلة الأساسية + بحث + PDF + فيديوهات + RTL.
- الإصدار التالي: إدارة محتوى عبر لوحة بسيطة، تعليقات سياقية، تحليلات استخدام متقدمة.
