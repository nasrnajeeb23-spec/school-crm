# دليل الإدارة المالية (Finance Guide)

نظام إدارة مالي شامل مصمم للمدارس لتنظيم الرسوم، الفواتير، والرواتب بكفاءة ودقة.

## 1. إعداد الرسوم (Fee Setup)
الخطوة الأولى والأهم هي بناء هيكل الرسوم.
*   **بنود الرسوم:** تعريف أنواع الرسوم (مثل: رسوم دراسية، رسوم تسجيل، كتب، زي مدرسي، باص).
*   **تخصيص الرسوم:** تحديد قيمة كل بند لكل مرحلة دراسية أو صف معين.
*   **العملة والضرائب:** النظام يدعم الإعدادات الضريبية وتعدد العملات (حسب إعدادات النظام).

## 2. إدارة الفواتير (Invoices)
*   **توليد الفواتير (Bulk Generation):** إنشاء فواتير لجميع طلاب مرحلة أو فصل معين بضغطة زر واحدة.
*   **الفواتير الفردية:** إنشاء فاتورة مخصصة لطالب معين.
*   **حالة الفاتورة:** متابعة الحالة (غير مدفوعة، مدفوعة جزئياً، مدفوعة بالكامل، ملغاة).
*   **الطباعة:** تصدير الفواتير وطباعتها بتنسيق رسمي يحتوي على شعار المدرسة.

## 3. تسجيل المدفوعات (Payments)
*   **استلام الرسوم:** تسجيل عملية دفع لطالب معين وربطها بفاتورة محددة.
*   **طرق الدفع:** تحديد طريقة الدفع (نقد، شيك، تحويل بنكي، دفع إلكتروني).
*   **سندات القبض:** إصدار سند قبض آلي عند تسجيل أي عملية دفع يمكن طباعته وتسليمه لولي الأمر.

## 4. الرواتب (Payroll)
*   **هيكل الراتب:** تحديد الراتب الأساسي والبدلات لكل موظف.
*   **قسائم الراتب (Salary Slips):** إصدار قسائم الرواتب الشهرية للمعلمين والموظفين.
*   **الخصومات والمكافآت:** إضافة خصومات (مثل: غياب) أو مكافآت إضافية قبل إصدار الراتب.

## 5. التقارير المالية
*   **تقرير الإيرادات:** عرض إجمالي التحصيلات خلال فترة زمنية.
*   **تقرير المتأخرات:** قائمة بالطلاب الذين لديهم مستحقات مالية متأخرة.
*   **المصروفات:** تتبع المصروفات التشغيلية للمدرسة (كهرباء، صيانة، إلخ) لحساب صافي الربح.
