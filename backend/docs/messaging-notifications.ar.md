# الرسائل والإشعارات (Messaging & Notifications)

نظام تواصل فعال يربط بين جميع أطراف العملية التعليمية (الإدارة، المعلمين، أولياء الأمور).

## 1. المحادثات (Conversations)
*   **محادثات فردية:** تواصل خاص وآمن بين المعلم وولي الأمر، أو بين المدير والمعلم.
*   **الخصوصية:** جميع المحادثات تتم داخل بيئة النظام لضمان الخصوصية والتوثيق.
*   **المرفقات:** إمكانية إرسال ملفات (صور، مستندات) داخل المحادثة.

## 2. الإشعارات (Notifications)
يتم إرسال إشعارات تلقائية في الحالات التالية:
*   **الحضور:** عند تسجيل غياب الطالب، يصل إشعار فوري لولي الأمر.
*   **المالية:** تذكير بموعد استحقاق الفاتورة أو عند تأكيد استلام الدفعة.
*   **الدرجات:** إشعار عند رصد درجات جديدة أو صدور الشهادات.
*   **النقل:** تحديثات حول حالة الحافلة المدرسية.

## 3. قنوات الإشعار
يدعم النظام قنوات متعددة (حسب إعدادات المدرسة):
*   **داخل النظام:** جرس التنبيهات في الشريط العلوي.
*   **البريد الإلكتروني:** ملخصات يومية أو تنبيهات فورية.
*   **الرسائل النصية (SMS):** للحالات العاجلة (تتطلب رصيد رسائل).
*   **إشعارات التطبيق (Push Notifications):** لمستخدمي تطبيق الجوال.

## 4. إنشاء رسالة جديدة
1.  اذهب إلى قائمة `الرسائل`.
2.  اضغط على `محادثة جديدة`.
3.  اختر المستلم (يمكن البحث بالاسم).
4.  اكتب الموضوع ونص الرسالة ثم اضغط `إرسال`.
