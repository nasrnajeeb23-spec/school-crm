# دليل المعلم — موسّع شاشة-بشاشة

## 1) لوحة المعلم
- مؤشرات: حصص اليوم، إشعارات جديدة، مهام معلّقة.
- روابط: جدول الأسبوع، فصولي، الرسائل.

## 2) الجداول (Timetable)
- شاشة الجدول:
  - أعمدة: اليوم، الوقت، الفصل، المادة.
  - إجراءات صحيحة:
    1. راجع حصص اليوم.
    2. استعد للحصص التالية عبر المواد المحددة.

## 3) فصولي (My Classes)
- شاشة الفصل:
  - تبويبات: الطلاب، الحضور، الدرجات، المواد.
  - الطلاب: قائمة بأسماء الطلاب مع فلاتر.
- الحضور:
  - حدّد حالة كل طالب (حاضر/غائب/متأخر/بعذر).
  - احفظ فورًا لتجنب فقدان البيانات.
- الدرجات:
  - أضف درجات لكل تقييم/اختبار.
  - تأكد من دقة الإدخال ومطابقته للمعايير.

## 4) الرسائل
- إنشاء/قراءة المحادثات الخاصة بالفصل أو مع الإدارة.
- إرسال مرفقات بشكل آمن وفق سياسات المدرسة.

## 5) إرشادات بلا أخطاء
1. حدّث الحضور مباشرة بعد كل حصة.
2. راجع الدرجات قبل النشر.
3. استخدم الرسائل للتوضيح عند تغييرات الجدول.
