# دليل ولي الأمر — موسّع خطوة بخطوة

## 1) الدخول
1. افتح رابط البوابة.
2. أدخل البريد وكلمة المرور (أو حدّث كلمة المرور عند أول دخول).

## 2) الصفحة الرئيسية
- بطاقات: آخر إشعارات، أحدث درجات، الحضور الأخير.
- روابط: ملف الطالب، الرسائل.

## 3) ملف الطالب
- معلومات عامة: الاسم، الصف، الفصل.
- الدرجات: قائمة التقييمات مع النتائج.
- الحضور: سجل حالات الحضور اليومية.

## 4) الرسائل
- تواصل مع معلم الفصل أو الإدارة.
- أرسل استفسارات ومرفقات عند الحاجة.

## 5) إرشادات بلا أخطاء
1. راجع الإشعارات يوميًا.
2. استخدم الرسائل لطلب التوضيح بدلًا من الاتصالات الهاتفية.
3. حافظ على بيانات الاتصال محدثة.
