# دليل النقل المدرسي (Transportation Guide)

نظام إدارة النقل المدرسي يتيح للمدرسة تنظيم حركة الحافلات وضمان سلامة الطلاب.

## 1. إدارة الحافلات (Buses)
*   **إضافة حافلة:** تسجيل بيانات الحافلة (رقم اللوحة، الموديل، السعة الاستيعابية).
*   **الحالة:** متابعة حالة الحافلة (نشطة، في الصيانة، خارج الخدمة).

## 2. إدارة السائقين (Drivers)
*   **بيانات السائق:** الاسم، رخصة القيادة، ورقم الهاتف للتواصل.
*   **ربط السائق:** تعيين سائق محدد لكل حافلة.
*   **تطبيق السائق:** يمتلك السائق تطبيقاً خاصاً لمتابعة المسار وتسجيل ركوب/نزول الطلاب (قريباً).

## 3. المسارات (Routes)
*   **إنشاء مسار:** تحديد نقطة البداية والنهاية والمحطات التي تمر بها الحافلة.
*   **توقيت الرحلة:** تحديد وقت الانطلاق والوصول المتوقع.
*   **تعيين الحافلة:** تخصيص حافلة وسائق لهذا المسار.

## 4. إدارة الطلاب (Students)
*   **الاشتراك:** تسجيل الطلاب في خدمة النقل وتحديد المسار المناسب لهم (ذهاب فقط، عودة فقط، أو كلاهما).
*   **قوائم الركاب:** طباعة قوائم بأسماء الطلاب لكل حافلة لضمان التنظيم.

## 5. تتبع ومراقبة
*   يمكن للمشرفين متابعة حالة الرحلات والتأكد من وصول الحافلات في مواعيدها.
*   إشعارات تلقائية لأولياء الأمور عند تحرك الحافلة أو وصولها (تتطلب تفعيل ميزة التتبع).
