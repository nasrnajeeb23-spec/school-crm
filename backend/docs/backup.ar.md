# النسخ الاحتياطي (Backup)

## الجدولة
- يتم ضبط المهام عبر إعدادات النسخ: يومي/شهري، وقت التنفيذ، أنواع البيانات.
- تنفيذ المهام عبر `node-cron` داخل الخادم.

## المسارات ذات الصلة
- إعدادات النسخ والجدولة والتنزيل: `backend/routes/schoolAdmin.js:1626–1713`.
- جدولة عبر `app.locals.scheduleBackupForSchool`: `backend/server.js:257–293`.
- تنظيف النسخ القديمة: `backend/server.js:308–326`.

## خطوات بلا أخطاء
1. فعّل إعدادات النسخ لمدرستك مع وقت مناسب.
2. راقب مجلد النسخ لكل مدرسة.
3. نزّل النسخة (zip) عند الحاجة.
